Learning Agents
What is a Learning Agent?

A Learning Agent is an AI system that can improve its performance over time by learning from experience.

Unlike previous agents that rely on fixed rules or predefined logic, a Learning Agent adapts its behavior based on feedback from the environment. 📈

This means the agent becomes better with every interaction.

Simple idea:

Act → Get Feedback → Update Behavior → Perform Better Next Time

How It Works

A Learning Agent continuously improves through a feedback loop:

[Act] → [Get Feedback] → [Update Weights / Knowledge] → [Act Better Next Time]
           ↑                       ↑
      "Was that good?"     "Adjust what matters"

Steps involved:

The agent takes an action.

It receives feedback about the result.

It analyzes whether the action was good or bad.

It updates its internal parameters or strategy.

Future actions become more accurate and effective.

Architecture of a Learning Agent

A Learning Agent typically contains four main components:

┌─────────────────────────────────────────┐
│  LEARNING ELEMENT    ← updates knowledge│
│  PERFORMANCE ELEMENT ← makes decisions  │
│  CRITIC              ← evaluates results│
│  PROBLEM GENERATOR   ← explores options │
└─────────────────────────────────────────┘
Components Explained

1. Performance Element
This part chooses actions based on the current knowledge of the agent.

2. Learning Element
It improves the agent’s knowledge or strategy using feedback.

3. Critic
The critic judges the outcome of the agent’s actions and provides feedback.

4. Problem Generator
This component encourages the agent to try new actions or explore new possibilities to improve learning.

Example

Music Recommendation System

At first, the system may recommend songs randomly.

But over time it learns from user behavior such as:

songs played

songs skipped

songs liked

Using this feedback, the system gradually improves its recommendations.

Why Learning Agents Are Powerful

Learning Agents are important because they:

Adapt to new environments

Improve automatically with experience

Handle complex and dynamic problems

Reduce the need for manual rule programming

"""
Learning Agent — Self-Improving Shopping Assistant
===================================================
Architecture:
  Performance Element  → makes recommendations (utility agent)
  Critic               → collects user feedback (thumbs up/down + reason)
  Learning Element     → updates weights based on feedback
  Problem Generator    → occasionally explores to avoid local optima

KEY UPGRADE over Utility-Based:
  Weights are NOT fixed. They shift after every feedback signal.
  The agent converges toward what the user actually values.

Learning Rule (Gradient-style weight update):
  If feedback is POSITIVE:
    Boost weights for criteria where chosen item scored high
  If feedback is NEGATIVE:
    Boost weights for criteria where chosen item scored LOW
    (i.e., learn what you should have prioritized instead)

Run:
  python learning_agent.py
"""

CRITERIA = ["price", "performance", "battery", "portability", "reliability"]


# ─────────────────────────────────────────────────────
# 2. AGENT STATE  (includes learned weights + history)
# ─────────────────────────────────────────────────────
def make_agent(budget: float = 2000, needs_gpu: bool = False) -> dict:
    """
    Initialize agent with equal weights (no prior knowledge).
    Weights will shift as the agent learns from feedback.
    """
    return {
        # Constraints
        "budget":     budget,
        "needs_gpu":  needs_gpu,

        # Weights — start equal, will be updated by learning element
        "weights": {c: 1.0 / len(CRITERIA) for c in CRITERIA},

        # Learning rate — how fast weights shift (0.0 = no learning, 1.0 = instant)
        "learning_rate": 0.15,

        # Memory
        "feedback_log":   [],    # list of {recommendation, feedback, scores, weights_before}
        "round":          0,
        "correct_streak": 0,

        # Exploration
        "explore_mode":   False,
    }


# ─────────────────────────────────────────────────────
# 3. UTILITY FUNCTIONS  (same as utility agent)
# ─────────────────────────────────────────────────────
def normalize(value: float, all_values: list, invert: bool = False) -> float:
    lo, hi = min(all_values), max(all_values)
    if hi == lo:
        return 1.0
    score = (value - lo) / (hi - lo)
    return 1.0 - score if invert else score


def score_laptop(laptop: Laptop, all_laptops: list) -> dict:
    """Return normalized scores (0–1) for each criterion."""
    return {
        "price":       normalize(laptop.price,       [l.price       for l in all_laptops], invert=True),
        "performance": normalize(laptop.speed_score, [l.speed_score for l in all_laptops]),
        "battery":     normalize(laptop.battery_hrs, [l.battery_hrs for l in all_laptops]),
        "portability": normalize(laptop.weight_kg,   [l.weight_kg   for l in all_laptops], invert=True),
        "reliability": normalize(laptop.brand_score, [l.brand_score for l in all_laptops]),
    }


def compute_utility(scores: dict, weights: dict) -> float:
    return sum(weights[c] * scores[c] for c in CRITERIA)


def get_feasible(agent: dict) -> list:
    return [
        l for l in CATALOG
        if l.price <= agent["budget"]
        and (not agent["needs_gpu"] or l.gpu)
    ]


def recommend(agent: dict) -> tuple:
    """
    Performance Element: pick the laptop with highest utility
    given current weights. With some probability, explore instead.
    """
    feasible = get_feasible(agent)
    if not feasible:
        return None, None

    # Exploration: occasionally try a random non-top choice
    agent["explore_mode"] = (
        agent["round"] > 0
        and agent["correct_streak"] < 2
        and agent["round"] % 4 == 0
    )

    scored = []
    for laptop in feasible:
        scores = score_laptop(laptop, CATALOG)
        utility = compute_utility(scores, agent["weights"])
        scored.append((laptop, scores, utility))

    scored.sort(key=lambda x: x[2], reverse=True)

    if agent["explore_mode"] and len(scored) > 1:
        # Pick 2nd best to explore — maybe it reveals new preferences
        chosen = scored[1]
    else:
        chosen = scored[0]

    return chosen, scored   # (best, all_ranked)


# ─────────────────────────────────────────────────────
# 4. LEARNING ELEMENT  (the core of this agent type)
# ─────────────────────────────────────────────────────
def update_weights(agent: dict, scores: dict, feedback: str) -> dict:
    """
    Learning Element — adjust weights based on feedback.

    Positive feedback (thumbs up):
      Reinforce criteria where this laptop scored HIGH.
      "You liked it → what it's good at must matter more."

    Negative feedback (thumbs down):
      Reinforce criteria where this laptop scored LOW.
      "You didn't like it → you care more about what it lacked."

    Uses a soft gradient update so weights stay bounded and sum to 1.
    """
    old_weights = dict(agent["weights"])
    lr = agent["learning_rate"]

    if feedback == "up":
        # Boost weights proportional to how high this laptop scored
        deltas = {c: lr * scores[c] for c in CRITERIA}
    else:
        # Boost weights for what this laptop lacked (low scores)
        deltas = {c: lr * (1.0 - scores[c]) for c in CRITERIA}

    # Apply deltas
    new_weights = {c: agent["weights"][c] + deltas[c] for c in CRITERIA}

    # Re-normalize so weights always sum to 1.0
    total = sum(new_weights.values())
    agent["weights"] = {c: v / total for c, v in new_weights.items()}

    return old_weights   # return for display


# ─────────────────────────────────────────────────────
# 5. CRITIC  (interprets feedback)
# ─────────────────────────────────────────────────────
REASON_TO_CRITERION = {
    "1": ("price",       "too expensive / good value"),
    "2": ("performance", "too slow / great performance"),
    "3": ("battery",     "poor battery / great battery"),
    "4": ("portability", "too heavy / very portable"),
    "5": ("reliability", "unreliable brand / trusted brand"),
}

def get_feedback() -> tuple:
    """Critic: ask user for structured feedback."""
    print("\n  Was this a good recommendation?")
    print("  [1] 👍 Yes, great pick!")
    print("  [2] 👎 No, not what I want")
    print("  [s] Skip (no feedback this round)")

    choice = input("  > ").strip().lower()

    if choice == "1":
        return "up", None
    elif choice == "2":
        print("\n  What was most wrong with it?")
        for k, (_, label) in REASON_TO_CRITERION.items():
            print(f"  [{k}] {label}")
        reason = input("  > ").strip()
        criterion = REASON_TO_CRITERION.get(reason, (None, None))[0]
        return "down", criterion
    else:
        return None, None


# ─────────────────────────────────────────────────────
# 6. DISPLAY HELPERS
# ─────────────────────────────────────────────────────
def bar(value: float, width: int = 20) -> str:
    filled = int(value * width)
    return "█" * filled + "░" * (width - filled)


def print_weights(weights: dict, old_weights: dict = None, title: str = "Current Weights"):
    print(f"\n  📊 {title}")
    print(f"  {'Criterion':<14} {'Weight':>7}   {'Bar':<22} {'Change':>8}")
    print(f"  {'─'*56}")
    for c in CRITERIA:
        w = weights[c]
        old = old_weights[c] if old_weights else w
        delta = w - old
        delta_str = f"+{delta:.3f}" if delta > 0 else f"{delta:.3f}" if delta < 0 else "  —   "
        arrow = " ↑" if delta > 0.001 else (" ↓" if delta < -0.001 else "")
        print(f"  {c:<14} {w:>6.1%}   {bar(w):<22} {delta_str:>7}{arrow}")


def print_recommendation(laptop: Laptop, scores: dict, utility: float,
                          agent: dict, all_ranked: list, explore: bool):
    tag = "  🔍 [EXPLORING]" if explore else ""
    print(f"\n{'─'*58}")
    print(f"  🏆 Recommendation  (Round {agent['round']}){tag}")
    print(f"{'─'*58}")
    print(f"  {laptop.name}")
    print(f"  ${laptop.price:,.0f}  |  {laptop.ram_gb}GB RAM  |  "
          f"{laptop.battery_hrs}h battery  |  {laptop.weight_kg}kg  |  "
          f"GPU: {'Yes' if laptop.gpu else 'No'}")
    print(f"\n  Criterion scores vs current weights:")
    print(f"  {'Criterion':<14} {'Score':>7}   {'Bar':<22} {'Weight':>7}")
    print(f"  {'─'*54}")
    for c in CRITERIA:
        s = scores[c]
        w = agent["weights"][c]
        weighted = s * w
        print(f"  {c:<14} {s:>6.2f}   {bar(s):<22} {w:>6.1%}")
    print(f"\n  Utility Score: {utility:.4f}")

    print(f"\n  All options (ranked):")
    for i, (l, _, u) in enumerate(all_ranked[:5]):
        medal = ["🥇","🥈","🥉","  4.","  5."][i]
        marker = " ← recommended" if l.name == laptop.name else ""
        print(f"  {medal} {l.name:<30} {u:.4f}{marker}")


# ─────────────────────────────────────────────────────
# 7. MAIN AGENT LOOP
# ─────────────────────────────────────────────────────
def run_learning_agent(agent: dict, rounds: int = 8) -> None:
    print(f"\n{'═'*58}")
    print(f"  Learning Agent — Adaptive Laptop Advisor")
    print(f"  Budget: ${agent['budget']:,}  |  Rounds: {rounds}")
    print(f"{'═'*58}")
    print(f"\n  Starting with equal weights (no prior knowledge).")
    print(f"  Give feedback each round — watch weights adapt!\n")

    print_weights(agent["weights"], title="Initial Weights (equal)")

    for round_num in range(1, rounds + 1):
        agent["round"] = round_num
        print(f"\n\n{'═'*58}")
        print(f"  ROUND {round_num} of {rounds}")
        print(f"{'═'*58}")

        # ── Performance Element: recommend ────────────
        result, all_ranked = recommend(agent)
        if result is None:
            print("  No feasible laptops found. Adjust budget.")
            break

        laptop, scores, utility = result

        print_recommendation(laptop, scores, utility,
                              agent, all_ranked, agent["explore_mode"])

        # ── Critic: collect feedback ───────────────────
        feedback, reason = get_feedback()

        if feedback is None:
            print("  ⏭  Skipped — weights unchanged.")
            continue

        # ── Learning Element: update weights ──────────
        old_weights = update_weights(agent, scores, feedback)

        # Update streak
        if feedback == "up":
            agent["correct_streak"] += 1
        else:
            agent["correct_streak"] = 0

        # Log it
        agent["feedback_log"].append({
            "round":     round_num,
            "laptop":    laptop.name,
            "feedback":  feedback,
            "reason":    reason,
            "utility":   utility,
            "weights_after": dict(agent["weights"]),
        })

        # Show weight update
        emoji = "👍 Positive" if feedback == "up" else "👎 Negative"
        print(f"\n  {emoji} feedback received — updating weights...")
        print_weights(agent["weights"], old_weights, title="Updated Weights")

    # ── Final summary ──────────────────────────────────
    print(f"\n\n{'═'*58}")
    print(f"  📈 Learning Summary — {len(agent['feedback_log'])} feedback rounds")
    print(f"{'═'*58}")

    # Show weight evolution
    initial = {c: 1.0 / len(CRITERIA) for c in CRITERIA}
    final   = agent["weights"]
    print(f"\n  Weight evolution (start → final):")
    print(f"  {'Criterion':<14} {'Start':>7}  {'Final':>7}  {'Change':>8}  Trend")
    print(f"  {'─'*52}")
    for c in CRITERIA:
        start  = initial[c]
        end    = final[c]
        delta  = end - start
        trend  = "↑ learned important" if delta > 0.03 else ("↓ learned less key" if delta < -0.03 else "→ stable")
        print(f"  {c:<14} {start:>6.1%}   {end:>6.1%}  {delta:>+7.1%}  {trend}")

    # What did the agent learn?
    top = max(final, key=final.get)
    low = min(final, key=final.get)
    print(f"\n  🧠 What the agent learned about you:")
    print(f"     Most important to you : {top} ({final[top]:.1%})")
    print(f"     Least important       : {low} ({final[low]:.1%})")

    # Final recommendation with learned weights
    result, all_ranked = recommend(agent)
    if result:
        laptop, scores, utility = result
        print(f"\n  🏆 Final recommendation with learned weights:")
        print(f"     {laptop.name}  —  ${laptop.price:,.0f}  (utility: {utility:.4f})")

def main():
    print("╔══════════════════════════════════════════╗")
    print("║   Learning Agent — Adaptive Advisor      ║")
    print("╚══════════════════════════════════════════╝")

    try:
        budget = float(input("\nYour budget in USD (e.g. 1500): $") or "1500")
    except ValueError:
        budget = 1500

    needs_gpu = input("Do you need a GPU? (y/n): ").strip().lower() == "y"

    try:
        rounds = int(input("How many rounds to learn? (e.g. 6): ") or "6")
    except ValueError:
        rounds = 6

    agent = make_agent(budget=budget, needs_gpu=needs_gpu)
    run_learning_agent(agent, rounds=rounds)


if __name__ == "__main__":
    main()
